# site_antidepressivos# Uso de Antidepressivos Pós-Pandemia no Brasil

## Visão Geral do Projeto

Este projeto de Big Data tem como objetivo investigar e analisar a evolução do consumo de antidepressivos no Brasil no período pós-pandemia de COVID-19. Utilizando um pipeline de dados robusto, modelos de Machine Learning e dashboards interativos, buscamos compreender as transformações nesse cenário e identificar padrões relevantes.

## Funcionalidades Implementadas

O site do projeto foi aprimorado com as seguintes funcionalidades:

*   **Blocos de Código Expansíveis:** Todos os trechos de código presentes no site agora contam com uma funcionalidade de expandir/recolher, permitindo uma visualização mais organizada e focada. Inicialmente, apenas um trecho do código é exibido, e o usuário pode clicar para ver o conteúdo completo ou recolhê-lo novamente.

*   **Visualizador de Dashboards Interativo:** Ao clicar em qualquer um dos dashboards, um visualizador em tela cheia (lightbox) é ativado, proporcionando uma experiência de visualização imersiva. Dentro deste lightbox, um botão "← voltar ao site" permite ao usuário retornar facilmente à página principal.

*   **Link para o Repositório GitHub:** Um link direto para o repositório oficial do projeto no GitHub foi adicionado ao rodapé do site, facilitando o acesso ao código-fonte e à documentação completa.

## Tecnologias Utilizadas

A stack tecnológica empregada no desenvolvimento e análise deste projeto inclui:

*   **Databricks:** Plataforma unificada para engenharia de dados, Machine Learning e análise.
*   **Scikit-learn:** Biblioteca Python para Machine Learning, utilizada na implementação de modelos de regressão linear.
*   **Prophet:** Ferramenta de previsão de séries temporais desenvolvida pelo Facebook.
*   **Power BI:** Ferramenta de business intelligence para criação de dashboards interativos.
*   **PySpark:** API Python para Apache Spark, utilizada no pipeline ETL.
*   **HTML, CSS e JavaScript:** Para a construção e interatividade do site.

## Estrutura do Projeto

O repositório contém a estrutura de um site estático, com os seguintes componentes principais:

*   `index.html`: O arquivo principal do site, contendo todo o conteúdo, estilos (CSS) e lógica de interação (JavaScript).
*   `assets/charts/`: Diretório com as imagens dos dashboards do Power BI.
*   `assets/docs/`: Contém documentos relacionados ao projeto, como apresentações e roteiros.
*   `assets/lib/`: Bibliotecas externas, como Highlight.js para colorização de sintaxe.
*   `assets/slides/`: Imagens dos slides da apresentação do projeto.

## Como Visualizar o Projeto

Para visualizar o site localmente, siga os passos abaixo:

1.  Faça o download ou clone este repositório.
2.  Navegue até a pasta `site/`.
3.  Abra o arquivo `index.html` em qualquer navegador web moderno (Google Chrome, Mozilla Firefox, Microsoft Edge, etc.).
